## aws organizations
recurso global que possui os seguintes recursos.

- gerenciamento centralizado de todas as contas.
- faturamento consolidado
- agrupamento hierarquico de contas
- politicas para centralizar o controle de serviços, tags e backups automáticos.
- integração com aws identity and access management.
- integração com outros serviços da aws.
